# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**, an industry-standard
code of conduct used by thousands of open-source projects.

The full text is published at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project (filing issues, submitting PRs, commenting
on discussions, etc.) you agree to abide by it.

## Reporting

If you experience or witness a violation, contact the project maintainers at
`christian.homberg@globalfishingwatch.org`. Reports are handled in confidence.
